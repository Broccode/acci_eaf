### Java/JDK Problems

```bash
# Check Java version
java -version
# Should show version 17 or higher

# Check JAVA_HOME
echo $JAVA_HOME

# Fix JAVA_HOME on macOS
export JAVA_HOME=$(/usr/libexec/java_home)
```
